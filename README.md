# Exam Python CMD521

Star Wars API menu app with database users.

## Setup

```cmd
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

Copy `.env.example` to `.env` and fill in your database credentials.

## Run

```cmd
python start.py
```

## Menu

1. Get all users (from database)
2. Films
3. People
4. Planets
5. Species
6. Vehicles
7. Starships
0. Exit
